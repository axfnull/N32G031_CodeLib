1、功能说明

    1、此例程展示通过外部触发中断，控制 LED 闪烁


2、使用环境

    /* 软件开发环境：当前工程使用的软件工具名称及版本号 */
    IDE工具：KEIL MDK-ARM 5.21.1.0
      
    /* 硬件环境：工程对应的开发硬件平台 */
    开发板：N32G031CL-STB V1.0     

3、使用说明

    /* 描述相关模块配置方法；例如:时钟，I/O等 */
    SystemClock：48MHz
    GPIO：PA5 选择作为外部中断入口，PB7 控制 LED(D3) 闪烁


    /* 描述Demo的测试步骤和现象 */
    1.编译后下载程序复位运行；
    2.按下松开 KEY2 按键，LED 闪烁；
